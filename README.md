# 💼 Portfolio de Youssouf Seme

- Python
- Flask
- HTML / CSS

## 📦 Installation

1. Cloner le projet
git clone https://github.com/tonusername/portfolio.git

2. Installer les dépendances
pip install -r requirements.txt

3. Lancer le projet
python app.py
